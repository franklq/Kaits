# Requirements

- Visual Studio
- SQL Server Express

# Variables WEB

Las siguientes variables son requeridas:

| variable name | sample value | description |
|------------|--------------|-------------|
|URL_BASE_API | http://localhost:14206 |  URL DEL API. |
|JWT_AUTH_TOKEN | ******* |  Clave para encriptacion del Token.  |
|JWT_AUTH_KEY | ******* |  Clave secreta del Usuario, debe ser igual a la variable del API.  |
